# TRACEAL Console

Tamper-Logging System - iOS Web App

## Files

- `index2.html` - Main application
- `styles2.css` - Exact Figma design styles
- `app2.js` - Application logic
- `logo.svg` / `logo.png` - Logo assets
- `icons.html` - IBM Carbon Icons reference

## Features

- Splash screen with TRACEAL branding
- User authentication with error handling
- iOS-optimized layout (375×812pt)
- Exact Figma specifications
- IBM Carbon icons (24×24px default)

## Typography

- Font: Söhne (Leicht 300, Buch 400, Kräftig 500)
- Colors: #E34949 (TRACEAL Red), #3B7FFF (Link Blue)

## View on iOS

Open `index2.html` in Safari on your iPhone.
